# Personal Portfolio Website (HTML + CSS)

## 📌 Project Overview

This project is a **Personal Portfolio Website** built using **HTML and CSS**.
It is designed to showcase a developer's profile, skills, and projects in a clean and responsive layout.

The website includes sections for introduction, about information, projects, and contact details.

---

## 🎯 Project Objectives

* Practice core **HTML structure**
* Learn **CSS styling and layout**
* Understand **Flexbox positioning**
* Build a simple **responsive web page**
* Create a beginner-friendly **portfolio template**

---

## 🛠 Technologies Used

* **HTML5** – for the structure of the website
* **CSS3** – for styling and layout design

---

## 📂 Project Structure

```
portfolio-project/
│
├── index.html      # Main HTML file
├── style.css       # CSS styling file
└── README.md       # Project documentation
```

## 🚀 Features
